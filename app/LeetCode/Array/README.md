# Array Problems

This directory contains LeetCode Array problems.

## Structure

Each problem should be in its own file following this pattern:

```php
<?php

declare(strict_types=1);

namespace App\LeetCode\Array;

/**
 * LeetCode Problem <Number>: <Title>
 * 
 * <Problem Description>
 * 
 * Time Complexity: O(?)
 * Space Complexity: O(?)
 */
class <ProblemName>
{
    public function solve(<typed parameters>): <return type>
    {
        // Implementation
    }
}
```
